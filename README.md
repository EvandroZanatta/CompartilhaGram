CompartilhaGram
===============

Cliente não oficial do Instagram para Firefox OS.
App para suprir a falta de um cliente oficial do Instagram no Firefox OS.


***


# Tecnologias empregadas

* HTML5
* CSS3
* JQuery
* Jquery/Ajax
* Jquery Mobile

# Requisitos Mínimos

* JQuery 1.11.0
* JQuery Mobile 1.4.0
